Simon Game Rules

Objective:
Repeat the color pattern shown by the game.

Gameplay:
Watch the sequence and click colors in the same order.

Scoring:
100 point for each correct step.

Levels:
Each level adds a new color to the sequence.

How to Play

step 1:
Press Start to begin.

step 2:
Watch the pattern carefully.

step 3:
Click the colors in the same order.

step 4:
Repeat as the pattern grows.
